# Sides of triangle 2

</div>

Nhập vào một số nguyên dương N. Kiểm tra xem có bao nhiêu bộ số a, b, c (a <= b <= c).<br>
<br>
Input:<br>
    - Dòng duy nhất chứa một số nguyên dương N (1 <= N <= 10^4)<br>
Output:<br>
    - Số lượng bộ ba (a, b, c) thoã mãn<br>
Giới hạn thời gian 1000ms<br>
