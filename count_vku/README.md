# Snaker Game (Contest 2)

</div>

Nhập vào một chuỗi S, kiểm tra xem chuỗi "VKU" xuất hiện bao nhiêu lần trong S.<br>
<br>
Input:<br>
    - Dòng duy nhất chứa chuỗi S(1 <= |S| <= 10^5)<br>
Output:<br>
    - Số lượng chữ VKU trong S<br>
<br>
Giới hạn thời gian 1000ms<br>

Test case mẫu:<br>

### Testcase 1:
|Input| Output|
|-----|:-----:|
|VKUabcVKU| 2|
